Turn off/on

Entire Extension:
Turning this off will turn of any feature of the extension. Good to have a fast way to see VVZ again as it is or if the extension suddenly breaks.
Adding CourseReview links:
Any CourseId (e.g. 252-0341-01L) on VVZ will be changed to link to their respective CourseReview page.
Adding ratings when searching for courses:
On the search page it will add the ratings from CourseReview in the table.
Everything for timetable creator:
Timetable creator is a feature that lets you save courses during search and on the mainpage you then see the amount of ECTS and you can create the timetable with exercise sessions and lectures.
Autofill in search menu:
This autofills the main search fields with the last entereed search. Makes it easier to look at different semesters and when you come back to VVZ after some time.
Filter when searching:
This let's you filter for the performance assesment (session exam, end of semester exam and (un)graded semesterperformance) while searching.
Searching by pressing enter:
In the search menu you can simply press enter anywhere and it will send the search.
Show old occurences of each course:
While on a course page it will query old occurences of that course (by course id) and will link them at the bottom of the page.
Directly link to course Overview page:
This modifies the link to the individual course pages such that it directly opens the overview.
Show all button even when ETH hides it:
The button after searching, to show all courses on the same page.
Information on hover in search results:
This will add fields to the search list, where you can get the information of the courses page right in the search list to quickly look over them.
